Tic Tac Toe

A Tic Tac Toe game created with HTML, CSS, and JavaScript on the frontend, and Python (Flask) on the backend. The frontend offers a visually appealing grid for user interactions, and JavaScript handles game logic. The Python backend, built with Flask, serves the web application. This combination provides a straightforward and interactive Tic Tac Toe experience for users.

Dice Roller

The DiceRoller game is a minimalist web application created using HTML, CSS, and JavaScript. With a sleek and intuitive design, users can roll two virtual dice, and the game determines the winner based on the higher rolled number. It's a quick and entertaining way to simulate a two-dice roll-off, making it an ideal choice for a simple and enjoyable gaming experience. The responsive interface ensures seamless interaction on different devices, adding a touch of excitement to the classic act of rolling dice.
